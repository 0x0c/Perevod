## Requirements

- C++11
  - I'm using `Apple LLVM version 7.0.0 (clang-700.0.72) Target: x86_64-apple-darwin15.0.0 Thread model: posix`.
- boost 1.59.0
  - boost.asio
- OpenCV 2.4.11 (for sample code.)

## Usage

```
make tc
```

or

```
make ts
```
